# Chimaera Sensor Unit

## About
This is the PCB design for the Sensor unit of the [Chimaera](http://open-music-kontrollers.ch/chimaera/) - a poly-magneto-phonic theremin.

## Prerequisites
To be used with [KiCAD](http://www.kicad-pcb.org).
